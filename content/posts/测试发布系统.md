这是一篇测试文章，用于验证博客发布系统的完整工作流程。我们已经成功创建了三个自动化脚本来简化博客文章的创建和发布过程。

<!--more-->

## 发布系统介绍

我们的博客发布系统包含三个核心脚本：

### 1. new-post.sh - 创建新文章

这个脚本可以快速创建新的 Markdown 文章，支持：
- 交互式输入标题、标签和分类
- 自动生成文章模板
- 可选的图片目录创建
- 保留中文文件名

### 2. word2md.sh - Word 文档转换

如果你习惯用 Word 写作，这个脚本可以帮你：
- 将 .docx 文件转换为 Markdown
- 自动提取文档中的图片
- 转换 TIFF 图片为 PNG 格式
- 修复图片路径引用

### 3. publish.sh - 一键发布

完成写作后，使用这个脚本可以：
- 自动检测文件变更
- 本地预览（可选）
- 自动 Git 提交和推送
- 触发 GitHub Actions 部署

## 使用示例

创建新文章：
```bash
cd ~/wzh_blog/scripts
./new-post.sh
```

发布文章：
```bash
cd ~/wzh_blog/scripts
./publish.sh
```
